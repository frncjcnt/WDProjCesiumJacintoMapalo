Pink Desk

Beginner

❌ No 3D Model

Step-by-Step Guide

Step 1: Foundation

First, lay out the foundation. As you can see from the image, use a variety of stone to create texture.
Tip: Mixing the stone makes it more authentic to medieval builds!
Fun Fact: Many medieval castles had secret tunnels and hidden rooms designed for escape or surprise attacks!

Step 2: Tower Walls

Using a combination of stone, stone bricks, and cobblestone, build up the walls of the tower as seen on the picture. Build up 8 blocks up. This already includes the base layer. Do this on all four sides of the tower. Scaffolding is the most recommend way to build up, however dirt blocks can also be used.

Step 3: Front & Back Design

Again, use a combination of stone, stone bricks, and cobblestone to create the front and back design of the house. Build up 3 blocks up. This already includes the base layer. Do this exact on both the front and back sides of the base. On the far right and left sides, add one block.

Step 4: Sideview

As for the sideview on the farthest side (away from the tower), this is the sequence of building up. Since we've already built the front and back design, you do not need to fill in the corner on this side. You start 5 blocks up (including the base layer). Increase this by 1 on the next part, and then increase it one more for the absolute peak (middle) of the sideview. The middle should be 7 blocks up (including the base layer). From the middle part, decrease it by 1 on the next part, and then finally decrease it back to 5 blocks up (including the base layer) on the far left side.

Step 5: Sideview (other side)

The other side is slightly unique, but it should follow the same building pattern in terms of building up. Check the image copy it as is. This is what the inside part of this wall should look like.

Step 7: Add the fences

For the first portion of adding fences, add them on the back end part of the tower, as shown in the image. From the ground, count 2 blocks up, and break 3 blocks above that to place the fences.

Fences: Front Tower
